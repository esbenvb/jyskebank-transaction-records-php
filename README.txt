PHP Library for generating transaction files for Jyske Bank

By Esben von Buchwald 2013, sponsored by anywear.dk

Documentation:
http://www.jyskebank.dk/wps/wcm/connect/1ccdeb0d-9587-44cd-87bb-f5b245e2cceb/Recordbeskrivelser_indl%C3%A6s.pdf?MOD=AJPERES

Status:
Currently the system only supports domestic transfers by check or bank-account. More options can be added easily in the same way as the JyskeBankDomesticTransfer class.

Usage:
See test.php for a usage examples and read the code :) Detailed documentation may come in the future :)
